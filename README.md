# 🥖 Loaf Life (by bby26)

Loaf Life is a mobile-first web application designed for students to find free or affordable campus-related resources like food events, student discounts, quick gigs, and creative life-hacks.

---

## 📌 Overview

Students often rely on scattered sources like bulletin boards or Facebook groups for useful, money-saving opportunities. Loaf Life brings these into one place via:
- A categorized interactive map.
- Peer-submitted deals, hacks and events
- Saveable content via user profiles.

## 🧩 Core Features (MVP)

### 🗺️ **Interactive Map**  
  Navigate a mobile-first map interface that shows free food events, student discounts, quick gigs and more.  
  - Add, edit, or remove pins in real time  
  - Filter by category (Food, Discounts, Gigs) and by distance  
  - Tap a pin to view full details and hit “Save” to bookmark  

### 💡 **Deals, Hacks & Events Feed**  
  Browse a scrollable list of peer-submitted posts formatted in easy-to-read markdown.  
  - Preview images, descriptions, and locations at a glance  
  - Save your favorites to a personalized “Pocket Cheatsheet”  
  - Use search or category filters to hone in on what you need  

### 🧾 **Submissions Portal**  
  Share your own tips / deals and events in just a few taps.  
  - Choose a category, and drop your pin on the map  
  - Describe the opportunity, set date/time, add some tags and hit “Submit”  
  - All submissions feed are dislpayed in the corresponding thread 

### 🔐 **User Authentication & Profile Dashboard**  
  Securely log in with email/password via Supabase Auth.  
  - View and manage your own submissions  
  - See and remove items you’ve saved  
  - Update your profile details and preferences

### 💾 **Save & Share**  
  Easily bookmark posts and map pins, then share your cheatsheet via link or QR code with friends.

## ⚙️ INSTALLATION

Follow these steps to get Loaf Life running locally:

    Clone the repository

git clone https://github.com/bby26/loaf-life.git  
cd loaf-life

Install dependencies

npm install  
# or  
yarn install

Set up your Supabase backend

    Create a free Supabase project at https://app.supabase.com

    In your project’s Settings → API, copy the URL and anon key

Configure environment variables
Create a file named .env.local in the project root with these entries:

NEXT_PUBLIC_SUPABASE_URL=https://your-project-ref.supabase.co  
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key

Run the development server

npm run dev  
# or  
yarn dev

Open http://localhost:3000 in your browser.

Build for production

npm run build  
npm start  
# or  
yarn build  
yarn start

Deploy to Vercel

npm install -g vercel  
vercel
